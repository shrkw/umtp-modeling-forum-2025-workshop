# 日程調整ソフトウェア

UMTPモデリングフォーラム2025ワークショップ「モデル駆動設計をやってみよう」で作成した日程調整ソフトウェアのコードリポジトリです。
https://shrkw.hatenablog.com/entry/2025/11/18/232645

----

候補日と参加者ごとの出欠を登録し、最も参加しやすい日付を見つける小さな Ruby プロジェクトです。

## クイックスタート

```bash
bundle install
bundle exec rake test  # 動作確認
```

使い方はminitestのテストコードを参照してください。

## 実装した要件

- 条件付きで参加を加味したスコアリング
- 合計が閾値以下なら `'再調整'` を返して再調整の必要性を示す
- 参加者の属性による重みづけ

## ステータスとスコア

| 記号 | 意味             | スコア | 備考                       |
| ---- | ---------------- | ------ | -------------------------- |
| ⚪︎   | 参加できる       | 1.0    | 必須参加者なら 2.0 に相当  |
| △    | 条件付きで参加可 | 0.5    | 必須参加者なら 1.0 に相当  |
| x    | 参加できない     | 0.0    | 必須/任意とも 0            |

各日付行で参加者スコアを合計し、もっとも高いスコアを持つ日付が候補になります。Role の重み付けにより必須参加者の影響を強めています。
